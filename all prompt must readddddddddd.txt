#the prompt

Create a full university SaaS website using Next.js (App Router).

Project Requirements:

1) Landing Page:
- Modern, premium, professional landing page.
- The company provides university lecture summaries in Question & Answer format.
- Include:
  - Hero section with strong headline, subtext, CTA buttons (Sign Up / Login)
  - Features section (Q&A summaries, 5 academic years, faster studying)
  - About section
  - Testimonials section
  - Footer
- Use suitable royalty-free images (Unsplash).
- Fully responsive (mobile-first design).
- Premium modern UI (Tailwind CSS + clean spacing + soft shadows + rounded cards).

2) Authentication (Advanced Version):
- Implement proper authentication (JWT-based).
- Use HTTP-only cookies for security.
- Create:
  - /auth/signup
  - /auth/login
- Store users securely (use mock database).
- Add password hashing (bcrypt).
- Add logout functionality.
- Protect routes properly.

3) Middleware Protection:
- Implement Next.js middleware.
- Block access to /dashboard if user is not authenticated.
- Redirect unauthorized users to /auth/login.
- Use secure token validation inside middleware.

4) Dashboard (Protected Page):
- Route: /dashboard
- Only accessible when authenticated.
- Add Tabs component with 5 tabs:
  - Year 1
  - Year 2
  - Year 3
  - Year 4
  - Year 5
- Each tab displays lecture summaries in structured Q&A format.
- Tabs should be interactive and responsive.
- Add logout button in navbar.

5) Production-Level SaaS Architecture:
- Proper folder structure:
  - app/
  - components/
  - lib/
  - middleware.ts
  - models/
  - services/
  - api/
- Use reusable components (Navbar, Tabs, AuthForm, Card, Button).
- Separate business logic from UI.
- Clean code and scalable structure.
- Environment variables for secrets.
- Error handling.
- Loading states.
- Form validation.

6) Premium UI Design:
- Modern SaaS style.
- Smooth hover effects.
- Clean typography.
- Responsive navbar.
- Mobile menu.
- Consistent color palette.
- Professional spacing system.

7) Technical Requirements:
- Next.js App Router.
- Tailwind CSS.
- No TypeScript errors.
- Clean readable code.
- Everything fully responsive for mobile, tablet, and desktop.

Generate all required files with proper structure and working logic.
